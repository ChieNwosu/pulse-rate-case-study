# Pulse Rate Case Study: Multiple Regression Analysis

A two-tool data science project combining **Python** (synthetic data generation) and **SAS Studio** (statistical analysis and modeling) to investigate physiological predictors of resting pulse rate.

---

## Project Overview

This case study applies multiple regression analysis to determine which physiological factors significantly predict resting pulse rate in adult subjects. The central hypothesis is that age and weight would show meaningful correlations with pulse rate.

**Research Questions:**
1. Which physiological factors significantly predict resting pulse rate?
2. What proportion of pulse rate variance can be explained by these factors?

---

## Workflow

```
Python (generate_data.py)
        │
        │  Generates 25 synthetic adult subjects
        │  with realistic physiological ranges
        ▼
pulse_rate_data.csv
        │
        │  Imported into SAS Studio
        ▼
SAS Studio (pulse_rate_analysis.sas)
        │
        ├── Descriptive statistics (PROC MEANS)
        ├── Normality testing (PROC UNIVARIATE)
        ├── Full regression model (PROC REG)
        ├── Reduced final model (PROC REG)
        ├── Interaction analysis (PROC GLM)
        └── Visualization (PROC SGPLOT)
```

## Dataset

**File:** `pulse_rate_data.csv`  
**Subjects:** 25 synthetic adults  
**Generated by:** `generate_data.py` using `numpy.random`

| Variable | Type        | Description                          | Units      |
|----------|-------------|--------------------------------------|------------|
| Age      | Continuous  | Subject's age                        | Years      |
| Weight   | Continuous  | Body weight                          | kg         |
| Height   | Continuous  | Height                               | cm         |
| Gender   | Binary      | 0 = Female, 1 = Male                 | —          |
| WBC      | Continuous  | White blood cell count               | ×10⁹/L    |
| RBC      | Continuous  | Red blood cell count                 | ×10¹²/L   |
| **Pulse**| Continuous  | **Resting pulse rate (target)**      | bpm        |

## Key Results

### Significant Predictors (α = 0.05)

| Predictor | Coefficient | p-value  | Interpretation                                      |
|-----------|-------------|----------|-----------------------------------------------------|
| Age       | −0.2564     | 0.0132   | Each additional year → −0.26 bpm in pulse rate      |
| Weight    | +0.3849     | 0.0004   | Each additional kg → +0.38 bpm in pulse rate        |
| WBC       | +1.7620     | 0.0100   | Each unit increase → +1.76 bpm in pulse rate        |

Gender, Height, and RBC were not significant and excluded from the final model.

### Final Model Equation

```
Pulse Rate = 49.0360 − 0.2564(Age) + 0.3849(Weight) + 1.7620(WBC)
```

### Model Fit

| Metric         | Value  |
|----------------|--------|
| R²             | 0.8277 |
| Adjusted R²    | 0.7702 |
| Root MSE       | 6.2590 |
| F-statistic    | 14.41  |
| p-value        | <.0001 |

The model explains **82.8%** of the variance in resting pulse rate.

## SAS Procedures Used

| Procedure       | Purpose                                              |
|-----------------|------------------------------------------------------|
| `PROC IMPORT`   | Load CSV into SAS dataset                            |
| `PROC MEANS`    | Descriptive statistics                               |
| `PROC UNIVARIATE` | Normality testing and histogram                    |
| `PROC REG`      | Full and reduced multiple regression models          |
| `PROC GLM`      | Age × Weight interaction analysis                    |
| `DATA step`     | Age group categorization (Young / Middle / Older)    |
| `PROC SGPLOT`   | Scatter plot with regression lines by age group      |

## How to Reproduce

### 1. Generate the data (Python)

```bash
pip install numpy pandas
python generate_data.py
```

This outputs `pulse_rate_data.csv`.

### 2. Run the analysis (SAS Studio)

1. Upload `pulse_rate_data.csv` to your SAS Studio file system
2. Open `pulse_rate_analysis.sas`
3. Update the file path in the `PROC IMPORT` step to match your upload location
4. Run the full program

## Limitations

- Small sample size (n = 25) limits statistical power and generalizability
- Synthetic data does not capture real-world physiological correlations between variables
- Future work should validate findings on a larger, real-world dataset
